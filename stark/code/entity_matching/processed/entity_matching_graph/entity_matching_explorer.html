- 点击节点会在左下角显示文本信息。
- 开启“展开全部实体值”后，点击商品节点会强制显示该商品全部实体节点（可用 Max Entity Values 限制）。
- 开启“遍历限制”后，category 节点不会通过入边回溯到大量 product，从而避免图爆炸。
